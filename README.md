# Vision Lab

Computer vision experiments focused on real-time detection, tracking and EO camera applications.

## Project Scope

This repository is used as a development workspace for:

- Real-time object detection pipelines  
- Multi-object tracking systems  
- UAV and EO camera vision applications  
- Edge AI deployment experiments  

## Target Use Cases

- Surveillance and ISR systems  
- UAV imaging applications  
- Electro-optical camera processing  
- Real-time decision support systems
