01 - FPS Overlay (Starter)

Goal:
Calculate and display basic FPS (Frames Per Second) on a live camera stream.

What you learn
- FPS concept (time-based calculation)
- Measuring time between frames
- Overlaying text on video frames
- Using time.time() for simple performance metrics

Key functions
- VideoCapture
- read()
- putText()
- imshow()
- waitKey()
- release()

How to run
python main.py
