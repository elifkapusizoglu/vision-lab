03 - Video Input Basics (Video Only)

Goal: Play a video file and display frames (no camera, no arguments).

What you learn
- VideoCapture (with a video file)
- read()
- imshow()
- waitKey()
- release()

How to run
python main.py

Notes
- Put a video file named "sample.mp4" in this folder.
- Press "q" to quit.
